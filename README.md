# the_sum_run_api
本repo旨在开发自动完成XXX跑的api
#### 基本功能：用户使用 Post 方法在body里面用如下json格式设置，即可完成跑步并且实现邮件通知。
```go
请求体样例  
{  
    "data": [  
            {  
                "username":"用户名",  
                "usercode":"用户imeicode",  
                "userdistance":"跑步距离",  
                "usermail":"xxx@qq.com"  
            }  
    ]  
}  
```
